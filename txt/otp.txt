経路探索サーバの開発環境
Python 2.7.18
Python 3.8.10
Openjdk-21-jdk
OpenTripPlanner2.5.0


1. 必要な依存パッケージをインストール
sudo apt update

2. Python 2.7.18 をインストール
調べてください

3. Python 3.8.10 をインストール
調べてください

両方の Python を確認
python2.7 --version
python3.8 --version

4. Javaのインストール
sudo apt install openjdk-21-jdk

5. その他パッケージをインストール
sudo apt install unzip
sudo apt install curl
pip install pandas

6. OTPインストール
wget https://repo1.maven.org/maven2/org/opentripplanner/otp/2.5.0/otp-2.5.0-shaded.jar

7. GTFSデータダウンロード・加工（formatted.py, upgrade_translations.py を /otp に配置してください）
curl -o sankobus.zip https://km.bus-vision.jp/gtfs/sankobus/gtfsFeed
curl -o dentetsubus.zip https://km.bus-vision.jp/gtfs/dentetsu/gtfsFeed
curl -o kumabus.zip https://km.bus-vision.jp/gtfs/kumabus/gtfsFeed
curl -o toshibus.zip https://km.bus-vision.jp/gtfs/toshibus/gtfsFeed
curl -o yatsushiro-taxi.gtfs.zip https://www.city.yatsushiro.lg.jp/kiji00315975/3_15975_115402_up_6k3sjd7i.zip
curl -o kumamoto-jr-kyushu.gtfs.zip https://gtfs-gis.jp/gtfs4research/gtfs-kumamoto-jr-kyushu-20240316.zip
curl -L -o kumamotoshiden.gtfs.zip "https://api.gtfs-data.jp/v2/organizations/kumamoto-shiden/feeds/kumamotoshiden/files/feed.zip?rid=current"
curl -L -o kumamotodentetsu.gtfs.zip "https://api.gtfs-data.jp/v2/organizations/kumamotodentetsu/feeds/kumamotodentetsu/files/feed.zip?rid=current"
unzip sankobus.zip -d sankobus
unzip dentetsubus.zip -d dentetsubus
unzip kumabus.zip -d kumabus
unzip toshibus.zip -d toshibus
python3 formatted.py sankobus
python3 formatted.py dentetsubus
python3 formatted.py kumabus
python3 formatted.py toshibus
python2 upgrade_translations.py sankobus
python2 upgrade_translations.py dentetsubus
python2 upgrade_translations.py kumabus
python2 upgrade_translations.py toshibus

8. 九州地方のOSMのデータを取得
wget https://download.geofabrik.de/asia/japan/kyushu-latest.osm.pbf

9. 熊本県の周辺を切り取る（南西：32.096306,129.923336、北東：33.1896,131.3318で設定しています）
sudo apt install osmctools
osmconvert kyushu-latest.osm.pbf -b=129.923336,32.096306,131.3318,33.1896 --complete-ways -o=kumamoto.pbf

10. OTPサーバー起動（otpディレクトリで実行する）
java -Xmx3G -jar otp-2.5.0-shaded.jar --build --serve .



OTPサーバーのバックグラウンド起動（ターミナルを閉じても起動したままにする場合）
1. tmux セッションを開始
tmux
2. OTPサーバー起動
java -Xmx3G -jar otp-2.5.0-shaded.jar --build --serve .
※セッションから切り離す
Ctrl + B を押した後、D を押す
※セッションに再接続
tmux attach


接続が完了の確認
http://localhost:8080/otp/routers/default/ 

入力GTFSデータ数の確認
http://localhost:8080/otp/routers/default/index/feeds
