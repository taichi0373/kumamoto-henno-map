# アプリケーション概要

## プロジェクト名

熊本県自主返納特典マップ

## 概要

熊本県自主返納特典マップは、熊本県の運転免許自主返納者向け特典情報を地図上で表示・検索できるWebアプリケーションです。自治体ホームページに分散している支援制度情報を一元化し、ユーザーの条件に応じた特典を分かりやすく提供することで、公共交通の利用促進と支援制度の効果的な活用を目指します。

## 背景・目的

### 社会的背景

- 地方の公共交通利用者数は減少傾向にあり、熊本県では昭和50年度から令和1年度にかけてバスの利用者数が約4分の1にまで減少している
- 2019年の池袋暴走事故を契機に運転免許証の自主返納が注目されたが、その後自主返納者数は年々減少している
- 支援制度の周知媒体として自治体ホームページが多く利用されているが、その内容がわかりづらく支援制度の認知を妨げる一因となっている

### 課題解決

- **情報の一元化**: 自治体ホームページに分散する特典情報を地図上で統一的に表示
- **条件別絞り込み**: ユーザーの居住地域・年齢・返納状況に応じた特典の自動抽出
- **位置情報の活用**: 視覚的に分かりやすい協賛店舗の位置表示
- **割引情報の表示**: GTFSデータを活用した公共交通での経路探索と割引運賃の表示

## 対象ユーザー

1. **運転免許返納者**
   - 年齢：主に65歳以上

2. **運転免許返納予定者**
   - 返納を検討中で事前情報収集をする層

3. **高齢者支援関係者**
   - 自治体職員、福祉関係者

## 主要機能

### エンドユーザー向け機能

#### 1. 地図表示・検索機能

- **インタラクティブマップ**: 熊本県内の特典提供施設を地図上にマーカー表示
- **現在地取得**: GPS情報による周辺特典の自動表示
- **ズーム・パン**: 地図の拡大縮小・移動による詳細表示

#### 2. 特典検索・絞り込み機能

- **キーワード検索**: 施設名・特典内容による検索
- **カテゴリ検索**: 店舗種別・サービス種別による絞り込み
- **条件別絞り込み**: 年齢・居住地域・返納状況による適用特典の抽出

#### 3. 特典詳細表示機能

- **施設情報**: 名称・住所・連絡先
- **特典内容**: サービス内容・適用条件
- **有効期限**: 特典の期限情報

#### 4. 経路案内機能

- **公共交通経路**: GTFSデータを活用したバス・電車での経路表示
- **徒歩経路**: 最寄り駅・バス停からの徒歩ルート
- **所要時間・運賃**: 移動時間と交通費の表示

## システム特徴

### 技術的特徴

- **レスポンシブデザイン**: PC・タブレット・スマートフォン対応
- **フロントエンド・バックエンド分離**: Vue 3 + Spring Boot による疎結合アーキテクチャ
- **RESTful API**: 拡張性と保守性を考慮したAPI設計

### データ

- **GTFSデータ**: 熊本県内の公共交通運行情報
- **地図データ**: OpenStreetMapベースの地図情報
- **自治体データ**: 各自治体の特典情報

## 期待効果

### 利用者メリット

- **情報アクセス向上**: 分散した特典情報への統一アクセス
- **利便性向上**: 地図ベースでの直感的な特典検索
- **移動支援**: 公共交通を活用した経路案内と割引運賃の表示

### 社会的メリット

- **自主返納支援制度の周知**: 自主返納者向け特典情報の提供により、制度の認知度向上
- **公共交通利用促進**: 公共交通の利用促進による地域活性化と環境負荷軽減
- **高齢者支援**: 移動制約のある高齢者の生活支援

## プロジェクト成果物

- **Webアプリケーション**: エンドユーザー向けフロントエンド
- **REST API**: 外部連携用API
- **データベース**: 特典・事業者・ユーザー情報管理
- **設計書・仕様書**: 本ドキュメント群
