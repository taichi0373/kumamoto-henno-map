## 画面別詳細仕様

### S001: ホーム画面（HomePage.vue）

**概要**: アプリケーションのメイン画面。地図表示と特典検索・経路案内機能を統合した中核画面。

**レイアウト構成**:
```
┌─────────────────────────────────────┐
│ ヘッダー（ナビゲーション・ユーザーメニュー） │
├─────────────────────────────────────┤
│                                     │
│            地図エリア                  │
│         （特典マーカー表示）             │
│                                     │
├─────────────────────────────────────┤
│ サイドバー（検索/経路案内/ユーザー特典タブ）│
└─────────────────────────────────────┘
```

**主要コンポーネント**:
- ヘッダーナビゲーション（AppHeader）
- 地図表示エリア（MapLibre GL JS）
- 特典検索タブ（AppSearchBenefit）
- 経路案内タブ（AppRouteGuidance）
- ユーザー特典タブ（AppUsersBenefit）

**画面要素**:
- **地図**: インタラクティブマップ、ズーム・パン操作
- **マーカー**: 特典施設のピン表示
- **サイドバー**: 特典検索・経路案内・ユーザー特典の3タブ切り替え
- **現在地ボタン**: GPS位置取得・表示
- **店舗表示切替**: マーカー表示・非表示の切り替え

**レスポンシブ対応**:
- **デスクトップ**: 地図とサイドバーの左右分割表示
- **スマートフォン**: 地図メイン、サイドバーはスライドアップ

### S011: ログイン画面（LoginPage.vue）

**概要**: ユーザー認証を行うログイン画面。

**レイアウト構成**:
```
┌─────────────────────────────────────┐
│            アプリロゴ                  │
├─────────────────────────────────────┤
│                                     │
│        ログインフォーム                 │
│     ┌─────────────────────────┐     │
│     │ ユーザー名               │     │
│     ├─────────────────────────┤     │
│     │ パスワード                │     │
│     ├─────────────────────────┤     │
│     │ [ ログイン ]             │     │
│     └─────────────────────────┘     │
│                                     │
│    新規会員登録はこちら                 │
└─────────────────────────────────────┘
```

**画面要素**:
- **ユーザー名入力**: ユーザー名フィールド
- **パスワード入力**: パスワードフィールド（マスク表示）
- **ログインボタン**: 認証実行
- **会員登録リンク**: 新規登録画面への遷移

**バリデーション**:
- 必須項目チェック
- パスワード最小文字数チェック

### S013: プロフィール画面（ProfilePage.vue）

**概要**: ユーザーの個人情報表示・編集画面。特典絞り込みに使用する属性情報の管理。

**レイアウト構成**:
```
┌─────────────────────────────────────┐
│ ヘッダー（プロフィール設定）             │
├─────────────────────────────────────┤
│ 基本情報                             │
│ ┌─────────────────────────────────┐ │
│ │ ユーザー名: [編集可能フィールド]     │ │
│ │ 生年月日: [日付選択]             │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 免許・居住情報                        │
│ ┌─────────────────────────────────┐ │
│ │ 居住自治体: [選択リスト]          │ │
│ │ 運転免許状況: [選択リスト]         │ │
│ │ 返納日: [日付選択]               │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ [ 更新 ] [ キャンセル ]              │
└─────────────────────────────────────┘
```

**画面要素**:
- **基本情報セクション**: ユーザー名、生年月日
- **居住情報**: 自治体選択（熊本県内市町村）
- **免許情報**: 所持状況（未所持・所持・返納・失効・その他）
- **返納情報**: 返納日（返納の場合のみ）
- **更新ボタン**: 変更内容の保存

## 画面間連携

### 認証フロー
1. 未認証ユーザーはゲストとしてホーム画面にアクセス可能
2. プロフィール設定機能は要ログイン
3. ログイン画面から認証後、元の画面に戻る
4. セッション切れ時は自動的にログイン画面に遷移

### 検索フロー
1. ホーム画面で検索条件を入力
2. 地図上にマーカー表示、サイドバーに結果表示
3. マーカーまたはリストアイテムクリックで情報確認
